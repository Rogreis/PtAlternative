### Revisão alternativa do Livro de Urântia para Português do Brasil

__Objetivos__: 
1. Usar o github com seus recursos para trabalhar numa revisão da tradução portuguesa do Livro de Urântia.
2. Criar um espaço para discussões sobre tradução. Por experiência, sabe-se que pensar em possíveis opções de tradução leva a um melhor e mais profundo entendimento do texto.
3. Permitir que mais pessoas possam ter acesso a este tipo de trabalho.
4. Acredita-se que o github tenha uma longa existência, pois é usado por milhões de pessoas no mundo de forma gratuita ou paga, o que deve dar longa vida vida ao trabalho aqui armazenado.

O texto do livro todo foi formatado em markdown, dividido em documentos (diretórios) e parágrafos (arquivos).

Com algum conhecimento de uso do github.com e do aplicativo git.exe, há 3 formas possiveis de colaboração:

 1. Fazer um fork do texto para sua conta github,
 2. Identificar a anotar um  problema de tradução.
 3. Clonar uma cópia em seu próprio computador
 4. Fazer parte da equipe que trabalha neste texto *(este é o meio mais difícil, pois a equipe já trabalha junto há muitos anos)*

**Fork**: é um recurso do github plenamente conhecido pelos seus usuários e fora de contexto ser explicado aqui.
**Anotação de problema**: crie uma anotação no link abaixo e ela será respondida no seu devido tempo:

		    https://github.com/Rogreis/PtAlternative/issues

Para **clonar** um repositório Git existem alguns pré requisitos:

1. Estar inscrito neste respositório como colaborador.
2. Conhecimento prévio mínimo do aplicativo git
3. Instalar, se ainda não tem, este aplicativo git. Pode obtê-lo aqui:  `https://github.com/git-guides/install-git`
4. Possuir algum tipo de terminal (prompt de comandos, powershell ou bash) instalado em seu computador com o git disponível no PATH.

Para saber se o git está instalado e configurado corretamente no PATH:
1. Abra o terminal
2. Digite o comando: `git -v`
3. Deve corretamente indicar a versão do git instalada.

Processo de clonagem:
1. Crie um diretório e abra o terminal nele.
2. Use o git para clonar: `git clone https://github.com/Rogreis/PtAlternative.git`
3. Navegue até o diretório do repositório recém-clonado: `cd PtAlternative`
4. Entre no branch "correcoes" onde se permite o envio de colaborações: `git checkout -b correcoes origin/correcoes`
5. Use seu editor de textos favorito ou mesmo um específico para markdown e modifique os parágrafos que deseje.


Ao final, de novo no terminal e estando no diretório onde clonou o respositório (PtAlternative):

`git add .`
`git commit -m "uma mensagem curta identificando o que fez"`
`git push`

Observação: se foram alterados diversos parágrafos, a mensagem acima vai ser associada a todos eles. É importante sintetizar bem o que foi feito, por exemplo:

        Correção do tempo do verbo
        Mudança de Espírito Materno para Espírito Mãe
        Sugestão para melhor entendimento
  
Para maiores instruções sobre o git: `https://www.w3schools.com/git/`  (em inglês)
