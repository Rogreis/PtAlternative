A semana seguinte foi dedicada a um programa de intenso treinamento. Cada dia os seis novos apóstolos eram confiados aos seus respectivos aliciadores para uma cuidadosa revisão de tudo o que tinham aprendido e experienciado em preparação para o trabalho do Reino. Os apóstolos mais antigos recapitularam cuidadosamente, para benefício dos seis mais recentes, os ensinamentos de Jesus até aquele momento. Ao anoitecer todos eles se reuniam no jardim de Zebedeu para receber a instrução de Jesus.
